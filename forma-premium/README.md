# RIVR — Fluid Asset Streams

A sleek DeFi dashboard Hero section built with React, Tailwind CSS v4, Motion, and Lucide icons.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** for fast bundling
- **Tailwind CSS v4** (`@tailwindcss/vite`)
- **Motion** (`motion/react`) for animations
- **Lucide React** for icons

## Project Structure

```
src/
  components/
    Hero.tsx           # Main hero section
    Navbar.tsx         # Top navigation bar
    HeroBadge.tsx      # "Fluid Staking" badge
    BottomLeftCard.tsx # Stats + Join Discord card
    BottomRightCorner.tsx # Documentation corner widget
  App.tsx
  main.tsx
  index.css            # Tailwind + custom font setup
```
